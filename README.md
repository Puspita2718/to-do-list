# 📝 To-Do List Web Application

A responsive and feature-rich To-Do List application built using **HTML**, **CSS**, and **JavaScript**. The app allows users to manage tasks with due dates, providing real-time UI feedback and localStorage support.

---

## 🔧 Features

- ✅ Add new tasks with optional due dates
- 📅 Highlight overdue tasks visually
- ✏️ Edit tasks inline
- ✅ Mark tasks as completed (with line-through effect)
- 🗑️ Delete tasks with a smooth fade-out animation
- 💾 Persistent storage using `localStorage`
- 📱 Responsive UI and modern gradient styling

---

## 🚀 Getting Started
### 📂 Project Structure

📁 To-Do-List/

### ▶️ How to Run
1. Download or clone this repository:
git clone https://github.com/yourusername/todo-list-app.git

2. Open `index.html` in your browser.

That's it! No additional setup required.
